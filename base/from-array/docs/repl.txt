
{{alias}}( buf, order )
    Converts an array to a one-dimensional ndarray.

    Parameters
    ----------
    buf: ArrayLikeObject
        Input array.

    order: string
        Memory layout (either 'row-major' or 'column-major').

    Returns
    -------
    out: ndarray
        Output array.

    Examples
    --------
    > var x = {{alias}}( [ 1, 2, 3 ], 'row-major' )
    <ndarray>
    > {{alias:@stdlib/ndarray/shape}}( x )
    [ 3 ]
